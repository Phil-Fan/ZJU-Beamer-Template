## Quick Start
1. 下载并安装 TeXLive；
2. 下载本模板文件；
3. 编写 ```.tex``` 文件；
4. 使用 ```XeLaTeX``` 编译（当前仅支持 ```XeLaTeX``` 方式编译）。

```shell
make # 编译
```

```shell
make init   # 删除所有辅助文件以及图片
make clean  # 删除辅助文件
make help   # 显示帮助信息
```
